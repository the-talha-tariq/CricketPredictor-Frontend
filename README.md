# 🏏 WTC Qualification Scenarios - Frontend

This is the **frontend** for the **World Test Championship Qualification Scenarios Calculator**, built with **React.js and Tailwind CSS**. It provides an interactive UI to view possible qualification scenarios based on match outcomes.

## 🚀 Features

✅ Fetches and displays **pending matches**  
✅ Allows users to select a team and simulate **qualification scenarios**  
✅ Displays **final WTC rankings** based on points and percentage  
✅ Generates an **HTML report** of valid scenarios  
✅ Built with **React.js and Tailwind CSS**  

## 🛠️ Tech Stack

- **React.js** - Frontend framework  
- **Axios** - API requests  
- **Tailwind CSS** - Styling framework
